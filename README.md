
# 🌾 Rural Classroom – Frontend

A responsive **multi-role educational platform UI** built using React and Vite.
This project simulates an online learning system with separate dashboards for students, instructors, and administrators.

---

## 🚀 Features

* 🔐 Role-based login system (Student / Instructor / Admin)
* 📊 Separate dashboards for each user role
* 🧭 Structured routing and navigation
* 🎨 Responsive UI using TailwindCSS
* ⚡ Fast performance with Vite
